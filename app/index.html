Launchpad for Macmillan Higher Education
May, 2012 - Present

Led a team of 4-8 developers to create Macmilland Higher Education's flagship online learning platform. This project required close collaboration with the client's internal design team and the buisness stakeholders using Agile methodology.

Launchpad was built as a Single Page Application using ASP.NET MVC, jQuery, and jQueryUI. We used a host of open source libraries to manage client-side routing and history (Crossroads), templating (Handlebars), unit testing (Jasmine), and over a dozen custom-built jQuery plugins for various UI elements.

On the server we used Unity for dependency injection, Microsoft Test for unit testing, and the full ASP.NET MVC stack. We used AppFabric extensively for distributed caching and Nginx as a reverse proxy in our infrastructure. The backend provided a unique challenge due to a buisness requirement of storing most of our data in a proprietary 3rd party No-SQL data store.

The Launchpad project went live in August 2013 with an initial user count of roughly 20,000 students.

Additionally I designed and built a Mobile-first web application as a prototype of the future direction for LaunchPad using AngularJS, SSAS, Bootstrap, and a Node.js-based build system with a ASP.NET Web.API backend.
