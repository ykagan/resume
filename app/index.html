Launchpad for Macmillan Higher Education
May, 2012 - Present

Led a team of 4 developers to create Macmilland Higher Education's flagship eLearning portal. This project required close collaboration with the client's internal design team and the buisness stakeholders using an Agile methodology.

Launchpad was built as a Single Page Application using ASP.NET MVC, jQuery, and jQueryUI. We used a host of open source libraries to manage client-side routing and history (Crossroads), templating (Handlebars), testing (Jasmine + PhantomJS), and over a dozen custom-built jQuery plugins that functioned as interactive UI elements.

On the server we used Unity for dependency injection, Microsoft Test and Nsubstitue for unit testing, and the full ASP.NET MVC stack. The backend provided a unique challenge of having to store most of our data in a proprietary 3rd party XML-based data store.

The Launchpad project went live in August 2013 with an initial user count of roughly 10,000 users.
